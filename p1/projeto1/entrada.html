Bem vindo ao projeto 1, da disciplina Linguagem de Programação IV - Internet
Projeto desenvolvido com o uso de HTML e CSS, com o objetivo de criar um site com 4 páginas, sendo elas:
Entrada - Aonde se encontra o texto de apresentação do projeto (esta pagina)
Âncoras - Aonde se encontra um texto com a utilização de âncoras para navegação entre os pontos do texto
Formulário - Se encontra um formulário com campos da tabela Oficinas, e demais campos do padrão HTML
Relatório - Se encontra um relatório com os dados(aleatorios por enquanto), para ser usado de relatorio da tabela designada.
Para navegar entre as páginas, basta clicar nos links no menu superior
A navegação entra as paginas esta sendo feito pelo uso do iframe
Estilização do site feita com o uso de CSS
Para voltar a página anterior, basta clicar no botão voltar no menu superior
